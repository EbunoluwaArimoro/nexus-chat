### **Project Description**
This repository houses the Lab Phase project for the **Product Management for Software Engineers** course. **Nexus Chat** is a high-performance, real-time messaging application designed to facilitate seamless team communication and instant data synchronization.

### **Product Management Strategy**
From a PM perspective, this project prioritizes the following **Core Pillars**:

* **⚡ User Experience (UX):** Optimizing for sub-100ms latency to ensure "instant" message delivery.
* **📈 Product Scalability:** Engineering the backend to handle high concurrent connections and real-time state management.
* **🔒 Security & Privacy:** Implementing robust JWT authentication and private channel logic to protect user data.

---

### **Technical Stack**

* **Frontend:** `React.js` (Component-based UI & State Management)
* **Backend:** `Node.js` & `Express.js` (RESTful API Orchestration)
* **Database:** `MongoDB` (NoSQL persistence for messages and user profiles)
* **Real-time Engine:** `Socket.io` (WebSockets for bidirectional communication)
